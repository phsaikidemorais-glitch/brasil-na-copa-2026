# 🇧🇷 Brasil na Copa 2026

Site premium desenvolvido em **HTML, CSS e JavaScript puro** sobre o Brasil na Copa do Mundo 2026.

🔗 **Acesse o projeto online:**  
https://brasil-na-copa-2026.vercel.app/index.html

---

## 🚀 Sobre o projeto

O **Brasil na Copa 2026** é um projeto front-end criado para portfólio, com foco em design moderno, experiência visual, organização de conteúdo e interatividade.

A ideia foi transformar um tema popular — a Seleção Brasileira rumo à Copa do Mundo de 2026 — em uma experiência digital premium, com páginas dedicadas a jogadores, agenda, história, curiosidades e contato.

O projeto foi desenvolvido para praticar e demonstrar habilidades em:

- HTML semântico
- CSS moderno
- JavaScript puro
- Layout responsivo
- Efeitos visuais e 3D
- Glassmorphism
- Organização de arquivos
- Deploy com Vercel
- Versionamento com GitHub

---

## ✨ Funcionalidades

O site possui:

- Página inicial com visual premium
- Página de jogadores com cards e filtros
- Página de agenda dos jogos
- Página sobre a história da Seleção Brasileira
- Página de curiosidades
- Página de contato
- Menu de navegação entre páginas
- Design responsivo para desktop, tablet e celular
- Animações e efeitos visuais com JavaScript
- Estrutura pronta para futuras atualizações

---

## 📄 Páginas do projeto

### `index.html`

Página principal do site, com apresentação visual do projeto e chamada para explorar o conteúdo.

### `jogadores.html`

Página com jogadores organizados por posição, cards modernos e informações individuais.

### `agenda.html`

Página com estrutura de agenda, jogos, horários e caminho da Seleção durante a Copa.

### `historia.html`

Página contando momentos importantes da história do Brasil nas Copas do Mundo.

### `curiosidades.html`

Página com fatos, números e curiosidades sobre a Seleção Brasileira.

### `contato.html`

Página de contato e apresentação do projeto como peça de portfólio.

## ▶️ Como rodar localmente

Clone o repositório:

```bash
git clone https://github.com/phsaikidemorais-glitch/brasil-na-copa-2026.git
```

Entre na pasta:

```bash
cd brasil-na-copa-2026
```

Rode um servidor local:

```bash
python -m http.server 5500
```

Abra no navegador:

```text
http://localhost:5500/index.html
```
